Complete Syllabus: Zero to Job-Ready

This page prevents missing pieces. Cisco exams give the certification spine; real jobs need operating systems, tickets, hardware, cloud, security, wireless, documentation and proof.

# Core Learning Blocks

Level 0 Foundations

  • Computer basics, OS basics, files, logs, terminal.
  • IP, DNS, DHCP, gateway, ports, protocols.
  • Windows and Linux survival commands.

Subnetting Mastery

  • Binary, CIDR, masks, usable range, broadcast.
  • VLSM, summarization, wildcard masks, IPv6 prefixes.
  • Daily timed drills until subnetting is automatic.

OSI Troubleshooting

  • Layer 1 cable, power, SFP, speed and duplex.
  • Layer 2 VLAN, MAC, ARP, trunk, STP.
  • Layer 3 routing, gateway, subnet, reachability.
  • Layer 4-7 ports, firewall, DNS, TLS, app.

Cisco Path

  • CCST Networking Foundation.
  • CCNA job-ready foundation.
  • CCNP Enterprise: ENARSI first, ENAUTO second.
  • CCIE Enterprise Infrastructure thinking.
  • Architect portfolio, not active CCAr.

Real Hardware

  • RJ45, cable tester, patch panel, rack layout.
  • Console cable, SFP, fiber safety, PoE, UPS.
  • Device replacement and change-window checklist.

Data Center Basics

  • Rack units, ToR, spine-leaf, out-of-band management.
  • iDRAC, iLO, IPMI, dual NICs, redundant power.
  • Hyper-V and VMware networking basics.

Security and Firewall

  • Firewall rules, NAT, DMZ, VPN troubleshooting.
  • RADIUS/TACACS+, 802.1X, MFA, logs, certificates.
  • Common ports and secure device management.

Wireless Troubleshooting

  • RSSI, SNR, channel overlap, 2.4/5/6 GHz.
  • Roaming, interference, AP placement, channel width.
  • Connected to Wi-Fi but no internet workflow.

Cloud Networking Bridge

  • AWS VPC, subnets, route tables, IGW, NAT Gateway, SG, NACL.
  • Azure VNet, NSG, route tables, VPN Gateway, Private Endpoint.
  • Hybrid BGP, site-to-site VPN, overlapping subnet problems.

# Proof Standard

ConceptSimple explanation, why it matters, real-world use.
LabTopology, commands, expected result, break-it task.
TroubleshootingSymptom, suspected layer, commands used, root cause, fix, verification.
PortfolioScreenshot, config file, notes, GitHub link, interview answer.

# Certification Version Control

Rule: exam details change. Every cert page must show official source, exam version, last checked date, price/duration where relevant, and what changed since last review.

Complete syllabus means certification plus field skills plus proof.
