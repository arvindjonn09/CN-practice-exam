The Path

Six Levels: Zero Basics → Architect Portfolio

Built like old stone, not cardboard. Each level has its own page with the full syllabus as a progress-saving checklist, hands-on labs mapped to real-world scenarios, and a growth check at the end. No level gets skipped — the foundation carries everything above it.

# Levels 1–5

CCST

Level 1 · CCST Networking Foundation — Beginner

The entry point. Standards, addressing, endpoints, infrastructure, diagnostics and security basics — proven through help-desk-style labs.

→
CCNA

Level 2 · CCNA — Job-Ready — Associate

The classic foundation: how the entire internet works. VLANs, OSPF, NAT, ACLs, security, automation basics. This level makes me employable as a network engineer.

→
CCNP

Level 3 · CCNP Enterprise — Professional

Primary path: ENARSI first for deep routing, then ENAUTO for automation. Routing brain first, automation second.

→
CCIE

Level 4 · CCIE-Level Lab Thinking — Expert

The 8-hour-lab mindset: predict the network's state before typing a show command, build large topologies from memory, troubleshoot with a calm, strict method.

→
ARCH

Level 5 · Network Architect Portfolio — Design Thinking

Not a certification target — a portfolio stage. Architecture decision records, design documents, business-aware trade-offs. The shift from “how do I configure it?” to “why this design?”

→
Why Level 5 is a portfolio, not a cert: Cisco has ended the CCAr path, so chasing it would be chasing a closed door. Architect-level thinking is still the goal — proven through design documents and decision records instead of an inactive certification path.

# The Toolbox

The simulators and tools this whole journey runs on. Details per level on each level's page.

Cisco Packet Tracer

The main lab environment for Levels 0–2. Drag-and-drop topologies, real IOS commands.

Free (Cisco Networking Academy)

GNS3

Emulates real Cisco IOS. The step up from Packet Tracer — full command set, real SSH for automation scripts.

Free & open source

EVE-NG

Professional emulation platform for CCNP/CCIE-scale topologies with dozens of devices.

Free (Community)

Cisco Modeling Labs (CML)

Cisco's official virtual lab platform with legal, current images — what the CCIE lab exam itself runs on.

Paid

Wireshark

Packet capture and analysis. Watching a TCP handshake turns theory into intuition. Used at every level.

Free

Cisco DevNet Sandbox

Free cloud-hosted real Cisco gear with API access — covers the automation topics with zero hardware.

Free

Python + Netmiko / Ansible

The automation stack from CCNA onward. Git for everything.

Free

Anki + subnetting drills

Spaced repetition for facts; daily subnetting practice until answers take under 15 seconds.
