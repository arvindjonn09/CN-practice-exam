Level 0: Absolute IT and Networking Basics

This page exists so the roadmap truly starts from zero. No Cisco command is useful until the basic language of computers, networks, IP addresses, ports, and troubleshooting is clear.

# Goal

Level 0 pass condition: I can explain a small network, assign IP addresses, test connectivity, use basic Windows and Linux commands, and document one broken-network ticket without guessing.

# Full Checklist

Computer and OS basics Foundation + Evidence Required
Internet and network basics Foundation
Survival commands Hands-on
Mini labs Proof

# Evidence Tracker — Computer and OS Basics

For every task below, write what you did in your own words, attach a screenshot, and optionally attach a short video. The browser saves this locally. For permanent proof, export the note and commit the screenshot/video to GitHub.

Important: Because this is a static GitHub Pages site, attached files are saved only in this browser when small enough. Large videos should be kept on your computer and uploaded to the GitHub repo manually.
